#  Financial Performance & Profitability Analytics

## Project Overview
This project analyzes the historical performance and risk characteristics of major cryptocurrency assets—Bitcoin, Ethereum, and Binance Coin—using Python, SQL, Power BI, and Excel.
The focus is on return analysis, volatility measurement, and asset-to-asset comparison to support data-driven financial insights.

## Objectives
1. Evaluate historical price performance of multiple crypto assets
2. Calculate daily returns, cumulative returns, and volatility
3. Compare risk vs return across assets
4. Prepare clean, analytics-ready data for visualization and reporting

## Tools & Technologies
1. Python (Pandas, NumPy) – Data cleaning, transformation, financial metrics
2. SQL – Analytical queries (planned)
3. Power BI – Interactive dashboards (planned)
4. Excel – Data validation and cross-checking

## Dataset Used
Historical cryptocurrency price data including:
1. Open, High, Low, Close prices
2. Trading Volume and Market Capitalization
3. Assets analyzed: Bitcoin, Ethereum, Binance Coin
